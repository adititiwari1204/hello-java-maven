Overview

The Hello Java Maven Project demonstrates how to build, package, and automate a basic Java application using Apache Maven and Jenkins within a Linux (Ubuntu) environment.
This project is designed to showcase the CI/CD workflow — starting from coding a simple Java program to automating its build and deployment through Jenkins.
